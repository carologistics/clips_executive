Writing a Plugin
################

**Goal:** Implement a custom CLIPS Plugin in C++.

**Tutorial level:** Intermediate

**Time:** 20 minutes

.. contents:: Contents
   :depth: 2
   :local:

Background
----------

Plugins can be used to access CLIPS environments via their C++ API, as detailed in the |APG|.
Note

Prerequisites
-------------

You will need the ``custom_action_interfaces`` package and the ``Fibonacci.action``
interface defined in the previous tutorial, :doc:`../Creating-an-Action`.
