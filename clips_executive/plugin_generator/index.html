ROS Interface Plugin GeneratorÔÉÅ

Source code on GitHub.

In order to interface with ROS directly without the need for generic introspection, the ROS2 CLIPS-Executive offers a plugin generator for individual interfaces (messages, services or actions).

Note that the Generic ROS Msg Plugin provides an alternative for ROS messages specifically, which works directly through introspection instead of relying on individual plugins for each message type.

Generic support for services and actions is not implemented yet, hence the plugin generator is needed in order to deal with services and actions.
