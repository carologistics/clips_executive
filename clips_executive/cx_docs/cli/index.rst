Command Line Interface (CLI)
============================

Source code on :source-master:`GitHub <cx_clips_cli>`.

The |CX| also provides an interactive terminal application for inspecting and controlling a running |CX| node over ROS 2.

It comes with the following features:

- **Log window:** Displays CLIPS output, also comes with basic scrolling and search features and allows to save or stream the output to a file for later inspection.
- **CLI prompt:** Allows uers to type CLIPS expressions and submits them to an active CLIPS environment. Supports some convenience features such as persistent history and auto-completion.
- **CLI command palette:** Provides additional features for interacting with the CX and configuration options for the CLI
