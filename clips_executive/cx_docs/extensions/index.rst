.. _extensions:
Extensions
==========

This section documents optional extensions that add advanced functionality to the project.

.. attention::

  Extensions are typically **not available as binary ROS packages**.
  This is due to dependencies that are only distributed via ``pip`` and that are not provided by the target distribution directly.
  For this reason, extensions require a **source build of the project** (see :ref:`source_installation`).
