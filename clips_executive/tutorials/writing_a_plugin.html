Contents

Background

Plugins can be used to access CLIPS environments via their C++ API, as detailed in the Advanced Programming Guide (PDF). Note

Prerequisites

You will need the custom_action_interfaces package and the Fibonacci.action interface defined in the previous tutorial, ../Creating-an-Action.
